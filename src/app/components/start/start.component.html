Before entering the „Display Factory“ use this checklist to make sure you have all relevant information at hand!

  • You need a SAP access
  • SAP order number is at hand
  • Apo planning has been done beforehand (min 3 months before the promotion starts)
  • Details like cost center, delivery addresses etc are available
  • Lead time for Displays up to 5 weeks after order placement (depends on product availability)
  • During the process of ordering you can save your unfinished order anytime
